# USB
USB-中山大学小学出校中心

- 三个网页app分别在三个文件夹当中，可以自行部署


# 部署步骤：
具体某些代码（甚至html）里面有中文显示【绿色页面服务器】【二维码服务器】先自行修改为你的服务器的IP

```
git clone https://github.com/sysuyyds/NoMoreUSC
```
三个文件夹app扔三台服务器上面（或者一台上面多个端口你喜欢就好）
```
cd 某个文件夹
```
启动
```
python3 main.py
```
